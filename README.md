# Место
Репозиторий для приложения проекта `Mesto`, включающий фронтенд и бэкенд части приложения со следующими возможностями: авторизации и регистрации пользователей, операции с карточками и пользователями.

## Стэк:
* HTML
* CSS
* JavaScript
* React
* Node.js
* Express.js
* Webpack
* БЭМ
